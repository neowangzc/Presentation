<div class="loop-popovers">
<div class="loop-card loop-card-env">
<label for="loop-none" class="loop-close">×</label>
<div class="loop-kicker text-emerald-300">The World</div>
<h3>Environment</h3>
<p>This is the small society where the simulation happens. It contains other agents, outside information, rules about who can see what, and events we inject into the world.</p>
<p>When an agent acts, the environment changes. The next round starts from this updated world.</p>
</div>

<div class="loop-card loop-card-agent">
<label for="loop-none" class="loop-close">×</label>
<div class="loop-kicker text-purple-300">The Actor</div>
<h3>LLMs Agent</h3>
<p>Think of this as a simulated person. It sees part of the world, remembers what happened before, thinks about its goal, and decides what to do next.</p>
<p>The LLM gives the agent a more human-like reasoning process, while the final action is still recorded clearly for comparison.</p>
</div>

<div class="loop-card loop-card-tools">
<label for="loop-none" class="loop-close">×</label>
<div class="loop-kicker text-sky-300">Extra Senses</div>
<h3>Tools</h3>
<p>Tools let an agent look beyond its prompt. It can call an API, crawl a page, or search the web when the experiment allows it.</p>
<p>This makes the agent less like a closed chatbot and more like an actor using information channels in a real platform.</p>
</div>
</div>
